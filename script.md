この形式のmarkdownファイルを元に、Nano bnana Proで画像生成​してください。
「#」単位で一つのスライド画像を作成してください。

# フォーマット
- アスペクト比: 16:9
- 解像度: 2K

# デザイン
- シンプルな白背景、黒文字、メインカラーは青、フラットーカラー
- イメージキャラクターとしてかわいい黒猫。
- 小物はワイン、本、ペン、ノートを必要に応じて使用すること。
- 見やすいスッキリした図解にしてください。
- 最後に文字を清書するように再生成してください。文字以外の要素は変更禁止です。

# 入力
各スライドの内容をそのままNanno Banana Proに渡すこと。

例)
```
以下の説明文を元に、見やすいスライド資料を作成してください。
・フォーマット
アスペクト比: 16:9
解像度: 2K
・デザイン
{デザインに記載されている内容そのまま}
・説明文
{各スライドに記載されている内容をそのまま記載}
```

# 出力
- 文字は基本的に日本語。
- outputsディレクトリを新規作成し、その中にYYYY-MM-DD_HH-MM-SSフォーマットでディレクトリを作成し、 その中にスライド画像を保存してください。
- 各スライド名は各見出しタイトルでお願いします。
- 画像生成時は各スライドを並列生成してください。
